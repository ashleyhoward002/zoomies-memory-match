# Zoomies Memory Match

An interactive memory-matching card game built with React and TypeScript, with type-safe game state management.

**Live demo:** https://zoomies-memory-match.vercel.app

## About

Zoomies Memory Match is a browser-based card matching game. Players flip cards to find matching pairs, with game state (flipped cards, matched pairs, move count) managed through typed React state.

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/ashleyhoward002/zoomies-memory-match.git
cd zoomies-memory-match
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Build

```bash
npm run build
```
